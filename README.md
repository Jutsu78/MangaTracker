**Project Theme:** A custom API designed to assist in managing and tracking a personal manga collection.

## Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database & ORM:** SQLite, Prisma ORM
* **Web Scraping (Planned):** Cheerio, Axios

## How to run/Як запустити ?


